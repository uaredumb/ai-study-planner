# CLAUDE.md

## Design Context

This project has captured design context for the `/impeccable` skill:

- **PRODUCT.md** — register (`product`), users, purpose, brand personality ("calm, clear, trustworthy"), anti-references, design principles.
- **DESIGN.md** — visual system tokens: colors, typography, elevation, components, do's/don'ts. North star: "The Calm Command Center."
- **.impeccable/design.json** — machine-readable design-system sidecar (tonal ramps, motion tokens, component snippets).
- **.impeccable/live/config.json** — live variant-mode config (multi-page static site, `*.html` at repo root).

Read PRODUCT.md and DESIGN.md before making UI/UX changes.
